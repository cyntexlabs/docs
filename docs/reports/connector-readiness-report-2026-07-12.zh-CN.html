最新代码证明什么，不证明什么

目前有代码证据的内容

catalog 已知 ID、规范化后的字段名/类型/默认值/选项、声明或推导出的 Source 模式，以及 Target 能力标记。

离线校验 CapabilityRules 会拒绝已知且可信的非法模式，以及类型或枚举不合法的配置值。

适配器接口 PDK adapter 可以隔离加载连接器类并读取注册能力，不会建立数据库连接。

当前代码不能建立的结论

连通性测试 UnavailableConnectionProbe 明确返回 NOT_IMPLEMENTED。

运行时成功 本报告中的连接器均未通过真实凭证、网络、权限、读写、TLS 或性能的运行验证。

离线局限 未知 ID 和未知配置键会延后；必填字段存在性也由服务端负责。一次本地 validate 成功不等于连接有效。

可执行代码的品牌/DSL 命名仍落后于文档计划。

最新产品代码仍使用 cyntex/v1、*.cyn.yml 和 cyntex CLI；而读者文档已经按计划使用 TapState 契约。客户要直接执行文档之前，必须先完成产品代码重命名，或明确给出一段兼容迁移期。
