# Held connector guides

These guides are intentionally excluded from the public documentation source until their supporting product feature and deployment flow are ready to release:

- Lark Bitable, Lark Approval, and Lark Doc
- Alibaba 1688
- Kafka (legacy)

To publish one again, move its MDX file (and any paired image assets) back into the public connector directories, restore it to `content/docs/connectors/meta.json` and `connectorDirectory`, then reclassify its catalog and docs-en mapping as published.
