## Current status

Do not create production resources from this page yet. The connector contract must define the final ID, Snapshot/CDC support, target capability, connection fields, and SQL Server CDC behavior.

## Upstream baseline

The reused connector documentation includes SQL Server Agent, database/table CDC enablement, retention, and schema-change constraints. Treat these as upstream preparation inputs rather than current TapState behavior.

After the catalog is available, migrate the source and target paths separately and verify all commands against the exact connector version and supported SQL Server editions.
