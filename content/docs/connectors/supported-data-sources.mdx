## Maturity labels

- **GA** means the connector has a stable, generally available documentation contract and a mature upstream baseline.
- **Preview** means the connector is ready to evaluate, but its configuration or behavior may still change before GA. Validate production workloads with the connector contract supplied by your TapState deployment.

TapState uses these two labels deliberately. The upstream inventory's former **Beta** group maps to **Preview**, a clearer customer-facing term. We do not show **Alpha** until a connector has a documented, limited-access contract.

## How to read roles and modes

- **Source** reads from the listed system. The read mode tells you whether that happens as a one-time Snapshot, Snapshot + CDC, stream, or API read.
- **Target** means the current connector catalog declares data writing. Check the connector guide before relying on table creation, schema evolution, or a particular write behavior.
- **Not declared** means the catalog has a source connection contract but does not declare a readable mode. A dash means the guide is target-only. Neither value implies an undocumented source mode.

<Aside type="note" title="Local validation boundary">
  `tapstate validate` checks the resource envelope plus catalog-known connector IDs, modes, and field shapes. It does not prove network reachability, credentials, permissions, API scope, data access, or a live write.
</Aside>

For setup and limitations, open the individual connector guide. The sidebar groups all guides into collapsible categories matching this directory.
