## Browse by category

The sidebar groups the directory into collapsible categories. Open the category that matches your integration task, then use the profile on each connector page for the exact roles and modes.

## Read the directory correctly

| Signal | Meaning |
|---|---|
| Maturity | **GA** is a stable, generally available contract. **Preview** is available for evaluation, but its configuration or behavior may change before GA. |
| Source | The documented way to read data: Snapshot, CDC, stream, or API. |
| Target | The current catalog declares data writing; details remain on the individual guide. |
| Schema changes | DDL handling only when it is explicitly evidenced; otherwise it is not claimed. |
