# Cyntex Docs

Cyntex is an Operational Data Hub for real-time CDC, YAML-defined data flows, connector-driven ingestion,
and AI-native control surfaces. Use this documentation to understand the product model, write valid
`.cyn.yml` resources, configure connectors, and connect AI agents through MCP.
