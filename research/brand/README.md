# Brand research archive

This directory preserves TapState brand research and source assets that are useful to product and design work but are not part of the documentation site's runtime surface.

- `proposals/` contains historical color and logo exploration.
- `logo/` contains source SVGs, favicon exports, and PNG exports.

Nothing under `research/` is published by the documentation site or referenced by application code. Runtime documentation assets belong in `public/` only when a page or application route directly references them. Application icons live in `src/app/`.
